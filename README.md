# KK Blog Skills

A collection of AI-powered "Skills" designed to optimize the content creation workflow for professional blogging, with a focus on AI product insights and WeChat Official Account formatting.

## 🚀 Skills Overview

### 🎨 Visual & Design
- **[blog-cover-generator](./blog-cover-generator)**: Professional blog cover image generator using HTML/SVG templates (900x363 resolution).
- **[blog-infographic-generator](./blog-infographic-generator)**: High-fidelity infographic and illustration generator for technical blog posts.

### ✍️ Writing & Editing
- **[blog-writer](./blog-writer)**: Structured AI assistant for drafting professional blogs about AI products and enterprise adoption.
- **[de-ai-writer](./de-ai-writer)**: Intelligent rewriter that removes "AI-sounding" patterns to produce natural, high-density Chinese content.
- **[insights-deepdive](./insights-deepdive)**: Socratic dialogue engine to extract non-obvious insights from raw ideas before writing.

### 🛠 Tools & Automation
- **[video-lens](./video-lens)**: Deep analyzer for YouTube videos that extracts transcripts, summaries, and timestamped insights.
- **[wechat-formatter](./wechat-formatter)**: Strict markdown-to-HTML formatter adhering to WeChat Official Account aesthetic rules.
- **[blog-organizer](./blog-organizer)**: Maintenance tool for standardizing blog Frontmatter and managing image assets.

## 🛠 Installation & Usage

These skills are designed to be used within the **OpenCode** environment. Each skill directory contains a `SKILL.md` file with specific instructions.

1. Clone this repository into your skills directory.
2. Ensure your AI agent has access to these paths.
3. Trigger skills by name or functionality (e.g., "Generate a blog cover").
